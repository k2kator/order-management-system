analysis module

class analysis.DataAnalyzer(customer_model, product_model, order_model)[исходный код]

Базовые классы: object

Класс для анализа данных и создания визуализаций. Обеспечивает построение графиков и диаграмм на основе данных о клиентах, товарах и заказах.

show_customer_network(parent_frame)[исходный код]

Строит граф связей между клиентами на основе общих заказанных товаров. Узлы - клиенты, связи - общие товары в заказах.

Параметры:

parent_frame – Родительский фрейм Tkinter для размещения графика

show_orders_dynamics(parent_frame)[исходный код]

Отображает динамику количества заказов по дням в виде линейного графика.

Параметры:

parent_frame – Родительский фрейм Tkinter для размещения графика

show_top_customers(parent_frame)[исходный код]

Отображает топ-5 клиентов по количеству заказов в виде столбчатой диаграммы.

Параметры:

parent_frame – Родительский фрейм Tkinter для размещения графика
