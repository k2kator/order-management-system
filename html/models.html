class models.BaseModel(filename, fieldnames)[исходный код]

Базовые классы: ABC

Базовый абстрактный класс для всех моделей данных.

Предоставляет общую функциональность для работы с CSV файлами: загрузка, сохранение, добавление, удаление и поиск элементов.
