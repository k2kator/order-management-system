Установка
==========

Требования
-----------

- Python 3.8+
- Tkinter (обычно входит в состав Python)
- Дополнительные зависимости: pandas, matplotlib, seaborn, networkx

Установка зависимостей
----------------------

.. code-block:: bash

   pip install pandas matplotlib seaborn networkx

Запуск приложения
-----------------

.. code-block:: bash

   python main.py
