Введение
---------

Система управления заказами - это десктопное приложение для управления клиентами, товарами и заказами.

Основные возможности:

- Управление клиентами
- Управление товарами  
- Создание и отслеживание заказов
- Аналитика и визуализация данных
- Импорт/экспорт данных

Быстрый старт
-------------

.. code-block:: bash

   python main.py
